记一次 Mac 安装 Pygame 报错的故事

Posted on 2019-11-12 Edited on 2020-03-10

环境

Mac OS 10.14.
Python 3.7.4
Pygame 1.9.6

问题描述

Mac 运行 Pygame 窗口空白，有声音，无图像。
运行python -m pygame.examples.aliens只能听到声音，看不到图像。

故障定位

python 与 pygame 不兼容情况导致。

解决方法

卸载本机 Python3.7.4 版本
卸载Python时会连以前下载的第三方库也删除掉，卸载后直接重装pygame即可。
卸载命令brew uninstall python3
卸载不掉则使用brew uninstall --ignore-dependencies python3强制卸载命令。

去官方下载 Python3.7.0 下载地址

https://www.python.org/downloads/release/python-370/

如果你本身就是 Python3.7.0 还遇到此问题。则更换Pygame版本
卸载本机 Pygame 1.9.6 版本。
卸载命令pip3 uninstall Pygame

安装 Pygame 1.9.4.dev0 版本
下载（重装）命令v

python3 -m pip install pygame==1.9.4.dev0

安装完成后测试图像是否正常。

python -m pygame.examples.aliens

小记

不需要安装什么虚拟环境！
不需要 Mac 降级！
不知什么是 brew 请自行Google
