Python的字符串方法

Posted on 2020-03-10

在 Python 中 Str 字符串类型的操作方法

基础方法

  • title() 将每个单词的首字母大写
  • lstrip() 只删除左边的空格
  • rstrip() 只删除右边的空格
  • strip() 用于移除字符串头尾指定的字符（默认为空格或换行符）
  • startswitch() 以参数开头开头
  • find 找子序列
  • replace() 将某个字符串中的某子序列替换成制定的值
  • upper() 全部变大些
  • isxxx() is开头的str方法，是数字吗？是xxxx码？特有功能。
  • str[3] 对字符串做索引，第3个下标的字符。
  • count() 计算字符串中，有几个相同字符。（需带参）
  • len() 计算字符串的长度
  • split() 字符串分割函数
